Este proyecto personal consistió en realizar mi portafolio CAS usando herramientas web, en este caso especifico me dedique a realizarlo con HTML (lenguaje de etiquetas) y CSS (lenguaje de estilos), con estas 2 herramientas me dedique a realizar mi proyecto de CAS.

Este proyecto me llevo mucho tiempo ya me costo aprender a escribir de manera correcta tanto las etiquetas como los estilos, para esto lleve mucho tiempo de investigación para cada lenguaje, pero aun me falta bastante por dominar esto sin contar que no use JavaScript, pero es un lenguaje que también domino, la razón por la que no lo use es que no sentí la necesidad de hacerlo, ya que la pagina solo funcionara como una especie de blog para colgar mis proyectos de CAS

Este proyecto fue mi favorito por todo lo que aprendí y mi profesión se desenvolverá en ser desarrollador web, específicamente un JavaScript Engineer, ese proyecto me motivo y pude ver mi gran avance de programación que tuve.
